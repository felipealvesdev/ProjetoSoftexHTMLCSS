O que é Javascript?


JavaScript é uma linguagem de programação imperativa baseada em eventos (em oposição ao modelo de linguagem declarativa do HTML) usada para transformar uma página HTML estática em uma interface dinâmica.
O código JavaScript pode usar o Document Object Model (DOM), fornecido pelo padrão HTML, para manipular uma página da web em resposta a eventos, como entrada do usuário.

O Javascript (abreviado em JS) é uma das linguagens de programação que, junto com HTML e CSS, formam as três principais tecnologias da World Wide Web.

É atualmente a principal linguagem para programação client-side em navegadores web. É também bastante utilizada do lado do servidor através de ambientes como o node.js.

Qual a função do Javascript?


Javascript serve um papel importante na criação de um site otimizado, utilizando de seus codigos que são inserido diretamente no HTML para serem interpretados pelo browser.
Fazendo uma experiência client-side melhor, e ainda tendo um leque de funções que não podem ser feitas no HTML e CSS.


Aqui umas vantagens de se usar o Javascript.


  • Menos interação com o servidor - Você pode validar a entrada do usuário antes de enviar a página para o servidor. Isso economiza tráfego do servidor, o que significa menos carga no servidor

  • Interoperabilidade - JavaScript é amplamente suportado e pode ser aplicado a qualquer página da web. Outras linguagens de script são muito menos flexíveis. Muitas linguagens de programação diferentes, como Pearl e PHP, também suportam JavaScript.

  • Feedback imediato aos visitantes - Eles não precisam esperar o recarregamento da página para ver se se esqueceram de inserir algo.

  • Versatilidade - JavaScript possui uma abundância de bibliotecas e frameworks: AngularJS, ReactJS, SvelteJS e outros. Essa versatilidade permite que os desenvolvedores criem uma variedade de tipos de aplicativos com atributos distintos.

  • Maior interatividade - Você pode criar interfaces que reagem quando o usuário passa o mouse sobre elas ou as ativa através do teclado.

  • Interfaces mais ricas - Você pode usar JavaScript para incluir itens como componentes de arrastar e soltar e controles deslizantes para fornecer uma interface rica aos visitantes do seu site.


Comandos mais utilizados no Javascript


Abaixo estarão os comandos mais frequêntes no Javascript

Comandos O que acontece?
alert() Mostra uma informação para o usuario no Browser via pop-up.
console.log() Informa via Debug uma informação solicitada.
prompt() Solicita o usuario que digite uma informação via pop-up.
function() Uma estrutura de comandos que pode ser chamada a qualquer momento num codigo.
if(){} / else{} Uma estrutura de condição que roda seus comandos se a condição for alcançada.
